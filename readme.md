# Toolbox : Java工具类库
## 简介
Toolbox是我在个人实际工作中常常用到的一些方法。最终目标是实现hutool和guava以及apache-commons中大部分常用的工具方法。设计理念是大部分常用的工具方法通过Toolbox对象操作，避免记各种工具类名。

实例代码:
```java
    //判断字符串为空
    Toolbox.isEmpty("");

    //打印格式化字符串
    Toolbox.println("this is {}", "toolbox");
```
## 依赖
- JDK17或更高
- 无其他依赖
